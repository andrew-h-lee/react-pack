# React-Pack App

My React Starter.

## Accomplished:

* Integrate multiple tools together.
* Developed a starter project from scratch.
* Basic starter project for future projects.

## Setup
```
npm install
```

## Compile
```
npm run compile
```

## Run Server
```
npm start
```
Open to `localhost:3002`
